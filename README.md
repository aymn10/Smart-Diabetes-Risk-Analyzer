# Smart Diabetes Risk Analyzer

A healthcare analytics platform that evaluates diabetes risk using clinical health indicators, risk scoring, and personalized recommendations.

## Project Type

* Healthcare Analytics
* Clinical Risk Assessment
* Predictive Health Monitoring
* Decision Support System

## Core Concepts

* Risk Scoring Engine
* Rule-Based Prediction System
* Health Trend Analysis
* Patient Assessment Tracking
* Data Visualization
* Personalized Recommendations

## Risk Factors Evaluated

* Blood Glucose
* Body Mass Index (BMI)
* Blood Pressure
* Age
* Insulin Level
* Skin Thickness
* Diabetes Pedigree Function
* Pregnancies

## Technology Stack

### Backend
